Additional commands:
	df-rootfs => rootfs ram usage statistics
	dpkg-forget-old-unavail => dpkg database cleaning tool
	load-drivers => load hardware drivers
	mount.cdrom.iso9660 => mount CD with ISO 9660
	mount.cdrom.udf => mount CD with UDF
	mount.cifs => load modules and mount samba share
	mount.ext4 => load modules and mount ext4 partition
	mount.nfs => load modules and mount nfs share
	mount.vfat => load modules and mount fat partition
	ntpdate => sync clock
	reboot-to-main => remove indicator and boot to main os
	swaps => print enabled swap devices
	umount-all => umount all filesystems except linux mountpoints
	zram-swap => enable compressed swap
