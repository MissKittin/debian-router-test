# webadmin
This package contains router modules for server-admin-page v3.1u2

### Extras
* notify-daemon: php orphaned sessions cleaner

### Installation
1. Run `share/webadmin/setup.sh`
2. Clone https://github.com/MissKittin/server-admin-page
3. Run `www/setup.sh` in cloned repo
4. Merge all files from `www` (from pt. 2) to `share/webadmin` (from pt. 1)
5. Run `install.sh` (and `install-extras.sh`, but it's not necessary) and follow instructions
